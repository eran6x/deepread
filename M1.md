# M1 — Foundations + Tier 1 Brief

This file documents what's in the M1 scaffold and how to run it.

## What ships in M1

- Repo scaffold: TypeScript, Vite + CRXjs, React 18, Tailwind v4, Biome, Vitest
- MV3 manifest with minimal permissions: `activeTab`, `storage`, `sidePanel`, `scripting`, plus a content script for `<all_urls>` (read-only listener; only does work when the side panel asks)
- Background worker with typed message router and a long-lived port for streaming analysis
- `LLMClient` interface with `DirectAnthropicClient` (BYO key)
- Anthropic SDK + tool-use streaming, prompt caching on the system prompt
- IndexedDB cache for analyses (Dexie), keyed by SHA-256 of extracted text
- Settings UI: paste API key, test it, masked display
- Side panel with **Brief** view rendering streamed verdict + brief + topics + est. read time + difficulty
- Mozilla Readability extraction in the content script
- Zod-validated `AnalysisResult` schema with partial validation for streamed updates
- Vitest unit tests for the schema

## Out of M1 (deferred to later milestones)

- M2: reader-mode rendering, span-to-DOM mapping, color-coded highlights, active-paragraph dimming
- M3: pacer, click-to-define, WPM/regression stats, sensitive-domain block-list, popup
- M4: edge-case sweep, alpha telemetry, distribution
- M5: hosted mode + billing (Beta)

## Prerequisites

This machine does not currently have Node.js installed. Install one of:

```sh
# Option A: Homebrew
brew install node@22 pnpm

# Option B: nvm or fnm or volta — any modern Node 20+ works
# then install pnpm:
corepack enable
corepack prepare pnpm@9.15.0 --activate
```

## Install + run

```sh
pnpm install
pnpm dev          # Vite dev server with HMR; load dist/ as unpacked extension
```

To load the extension:

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select the `dist/` directory created by `pnpm dev`
4. Pin the Deepread icon to the toolbar
5. Click the icon → side panel opens
6. Switch to **Settings**, paste your Anthropic API key, click Save
7. Open a long-form article (e.g. an Atlantic or Substack post)
8. Switch back to **Brief**, click **Analyze this page**

## Build

```sh
pnpm build        # tsc -b && vite build → dist/
pnpm typecheck    # tsc -b --noEmit
pnpm lint         # biome check
pnpm test         # vitest run
```

## Exit criterion (M1)

> "I can paste my own API key, hit Deepread on an NYT or Atlantic article, and see a streamed verdict + brief in <8 seconds, twice in a row (second one served from cache)."

Verify by:

1. Loading the unpacked extension
2. Saving a real API key in Settings; confirming the test call succeeds
3. Opening an article; clicking Analyze; observing verdict streams within ~3s and brief within ~5s
4. Refreshing the side panel and re-analyzing the same article; observing instant result with **"Loaded from cache"** phase line
